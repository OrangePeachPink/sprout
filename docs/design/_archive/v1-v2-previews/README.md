# v1 / v2 preview screenshots — archeology only

**Frozen originals. Nothing here is live.** These are the original rendered preview screenshots that shipped
with the early design deliveries — kept as a point-in-time visual record of how v1 and v2 actually looked.

- **`v1/`** — from the repo's original `docs/design/screenshots/` (the v1 design-system previews:
  `top` cover, `personality` / `personality-dry` mood system, `ladder` calibration range).
- **`v2/`** — from the repo's original `docs/design/sprout-v2/screenshots/` (the v2 brand delivery previews:
  deck cover, day-in-the-life, loader variants, the social set, retro, plus the shared `ladder` / `personality`).

## Where the live versions are

- **Current thumbnails:** every active page has a fresh preview in
  [`library/thumbs/`](../../library/thumbs/), surfaced on the Sprout Design Library.
- **The source they depict is preserved:** the v1/v2 page sources live across the shelves
  (`foundations/`, `brand/`, `voice/`, `motion/`, `go-to-market/`) and, pre-alignment, in
  [`../pre-v3-originals/`](../pre-v3-originals/).

## Why kept

Per *supersede, never delete* (ADR-0010): these previews were dropped from the active tree during the
shelf reorg, but the originals were preserved here so no provenance is lost. They are derived artifacts —
the source that produced them is intact — so treat them as archeology, not a dependency.
