Lesson from building Sprout: the percentage was lying.

Cheap soil sensors output raw ADC counts. Mapping that to a clean “83%” looks trustworthy — and isn't. I switched to honest raw values + 7 calibrated bands, and the watering got dramatically more reliable.

Full write-up and code in the repo. Calibration thread below.
