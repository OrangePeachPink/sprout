The range & movement band-lane view is the live counterpart to the calibration ladder: the same seven bands, same order, same --band-* tokens — but reading a chosen window instead of a static config. This page rules the visual language Data builds against: the marker set, the range whisker, the movement connector, the window selector, and the honest empty & excluded states. Every choice is weighed against one test: does it claim more than the calibration can back?

Marker set — the ruling

reconciles the one-off · R2 · R4 · R7

The one-off reached for filled/open circle, triangle and diamond. Ratified into five marks, each with exactly one job — so a shape never means two things, and no mark ever claims a reading we don't have.

Now dot — filled circle
Current band, from a cal_verified=true reading. Filled in the band's own color. Exactly one per entity.
Provisional ring — open circle
Same “now” role, but cal_verified=false. The hollow centre is the honesty signal — a lane we can place but not yet certify. Never earns a reach or connector.
From triangle — hollow triangle
Where the entity started this window. The apex points toward the now dot — that's the direction of travel. Hollow, so it never reads as a live reading.
Reach diamond — hollow diamond, ×2
Caps the reach field at the wettest and driest bands touched. A distinct shape on purpose — an extent mark, not “a reading is here now.”
Movement connector — dotted, stepped
Links observed positions from-dot to now. Dotted, never solid — a transition between two readings, not a continuous path. See below.
hard rule
No mark ever lands on the Parched (air-dry) lane for an unwired or NO_SIGNAL entity. Air-dry doubles as “probe may not be in soil” — a dot there would fabricate “bone-dry.” Those entities get no mark at all.

Window selector

R5 · reuses the range vocabulary

One segmented control, the same words the rest of Sprout already uses for ranges. Two anchors: a short event (~15 min — watch one watering play out) and a long horizon (a dry-down cycle).

15 min 1 h 24 h 7 d since last re-water
event · 15 min
Dry → watered → diffusing → resettling to a new moist median. Marks are dense; connector shows each observed step.
horizon · 24 h / 7 d
A full dry-down. Reach field is the story; the connector reduces to the net from → now move.
provisional · since last re-water
Needs a watering-event stream that doesn't exist yet. Ships disabled with a dot until that anchor lands — never guessed from a wet jump.

Honest empty & excluded states

R7 · R8 · consistent with #486

An unearned mark is the one thing this view must never draw. Four ways a row can have nothing honest to place — each says so plainly, in Sprout's voice.

The entity is wired and calibrated — it simply hasn't reported inside the chosen window yet.
Lanes stay visible so the scale is legible; a quiet line says “no readings in this window yet.”
EXCLUDED · per-device fence
belongs to another device group
An entity belongs to exactly one device — no cross-device roll-up onto shared lanes.
Filtered-out entities are named as out-of-scope, never silently merged into another device's picture.

Guardrails

what stays true no matter how the live view is built
1
Bands are the only scale. Raw + band words, never a %. The lanes carry wet→dry; a mark's job is position and identity, not a number.
2
Never claim an unobserved reading. Reach snaps to whole bands; movement is dotted and discrete; empty is empty. Precision we don't have is precision we don't draw.
3
Air-dry is ambiguous, so it's diagnostic. A NO_SIGNAL or unwired entity earns no mark — never a false “bone-dry” on the Parched lane.
4
One device, one fence. Entities never roll up across devices; the shared lanes are a scale, not a merge.
